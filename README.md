# website

My personal website, built with Hugo SSG.

It can be viewed at [lucwallace.com](https://lucwallace.com).

## Design

Design is heavily inspired by the [Swiss style](<https://en.wikipedia.org/wiki/Swiss_Style_(design)>) of graphic design.

The font used is [Neue Montreal](https://pangrampangram.com/products/neue-montreal), a grotesque typeface.
